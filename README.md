# 📦 Controle de Estoque em Excel

Projeto desenvolvido em Microsoft Excel para gerenciamento e acompanhamento de estoque. A planilha permite controlar entradas e saídas de produtos, cadastrar fornecedores e acompanhar indicadores por meio de dashboards e tabelas dinâmicas.

## 📋 Funcionalidades

- Cadastro de produtos
- Cadastro de fornecedores
- Registro de entradas de estoque
- Registro de saídas de estoque
- Controle da movimentação de produtos
- Análise de custos por período
- Dashboards com indicadores
- Tabelas dinâmicas para análise dos dados

## 📁 Estrutura da Planilha

| Aba | Descrição |
|------|-----------|
| **Produtos** | Cadastro dos produtos disponíveis no estoque. |
| **Fornecedor** | Cadastro dos fornecedores. |
| **Entradas** | Registro das entradas de produtos no estoque. |
| **Saídas** | Registro das saídas de produtos. |
| **Saídas por Produto** | Consolidação das movimentações por produto. |
| **Quantidades por Mês** | Análise da movimentação mensal em quantidade. |
| **Custo por Mês** | Indicadores de custos por período. |
| **Vendas por Produto** | Desempenho dos produtos vendidos. |
| **Dinâmica Relacionada** | Base de tabelas dinâmicas utilizadas nos dashboards. |
